# 📘 Exploration of Python Libraries

## 📌 Project Overview

This project focuses on exploring and understanding some of the most widely used Python libraries for data analysis and visualization. The notebook demonstrates practical implementation, examples, and outputs to strengthen foundational concepts in Python programming.

---

## 🎯 Objectives

* Understand the importance of Python libraries
* Perform data manipulation using Pandas
* Execute numerical operations using NumPy
* Create visualizations using Matplotlib and Seaborn
* Interpret results and insights from data

---

## 🛠 Libraries Used

* **NumPy** – Numerical computations
* **Pandas** – Data manipulation and analysis
* **Matplotlib** – Data visualization
* **Seaborn** – Statistical data visualization
